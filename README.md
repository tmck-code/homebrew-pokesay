## How do I install these formulae?

- Install directly
    ```shell
    brew install tmck-code/taps/<formula>
    ```
- Or
    ```shell
    brew tap tmck-code/taps
    brew install <formula>
    ```
- Or, in a `brew bundle` `Brewfile`:
    ```ruby
    tap "tmck-code/taps"
    brew "<formula>"
    ```
